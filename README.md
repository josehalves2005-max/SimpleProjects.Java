## 📌 Sobre este Repositório
Este espaço funciona como um portfólio de evolução técnica, onde cada projeto resolve um desafio de lógica aplicando as melhores práticas de desenvolvimento.



## 📂 Projetos em Destaque

### 1. 📊 Analisador de Dados Biométricos
Algoritmo para processamento estatístico de grupos (Alturas e Gêneros).
* **Destaques Técnicos:** Uso de vetores estáticos, manipulação de `Scanner` e tratamento de exceções aritméticas.
* **Complexidade:** $O(n)$ - Processamento de passagem única.



## 🛠️ Stack Técnica
* **Linguagem:** Java 17
* **IDE:** Apache NetBeans
* **Controle de Versão:** Git

## ⚙️ Como Executar
1. Clone o repositório.
2. Abra o projeto no **NetBeans**.
3. Localize o arquivo principal na pasta `src`.
4. Clique com o botão direito e selecione **Run File**.
