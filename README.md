# SimpleProjects.Java ☕

Repositório dedicado ao estudo e prática de **Lógica de Programação** e **Programação Orientada a Objetos (POO)** com Java. Aqui documento minha evolução técnica, desde algoritmos fundamentais até sistemas simulados.

## 🚀 Projetos Principais

* **Sistema Bancário:** Aplicação prática de encapsulamento, construtores e métodos de acesso (`getters`/`setters`).
* **Analisador de Dados Biométricos:** Algoritmo para processamento de estatísticas (médias, filtros de gênero e comparações).
* **Class Projects:** Coleção de exercícios de lógica e fundamentos de Java.

## 🛠️ Tecnologias
* **Java JDK 17+**
* **NetBeans / VS Code**
* **Git & GitHub**

## 📂 Como usar este repositório
1. Clone o projeto: `git clone https://github.com/josehalves2005-max/SimpleProjects.Java.git`
2. Importe na sua IDE favorita.
3. Explore as classes dentro de `src` para ver as implementações de POO.
